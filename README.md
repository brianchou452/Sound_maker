# ⼈聲合成器 — ⾳癡哼歌的好幫⼿  
* 資訊之芽大作業一
* 這是一個可以用自己的聲音和簡譜，產生wav檔的程式。這個程式可以幫助音癡哼歌，因為是透過程式改變原始音檔的頻率，所以不怕音調不準的問題。這個程式還擁有自訂音樂庫的功能可以輸入自己喜歡的音樂簡譜這樣就不必每次都要輸入一次簡譜。可以節省很多時間
